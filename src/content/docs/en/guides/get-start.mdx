## What is Bob-Ddong-Iri-Hoyo?

Bob-Ddong-Iri-Hoyo is a Wine launcher for running Eternal Return and HoYoverse games on macOS.
Internally, it uses a Wine-based runtime, and on Apple Silicon Macs it may also rely on compatibility layers such as Rosetta 2 and DXMT.

The goal of the launcher is to let users choose and manage the Bottles, Wine versions, and graphics translation layers needed to run games without directly handling complex Wine commands or configuration files.
Users who are already familiar with Wine can still use a CLI environment directly, but most users can perform the same tasks more easily through the launcher.

## Features

It addresses long-standing rendering issues that can appear when running Eternal Return through conventional CrossOver or Wine setups.

## Documentation Structure

This documentation is written to be read in the following order.

- **Installation**: Explains how to download the launcher and prepare it to run on macOS.
- **About Wine**: Describes the Wine builds provided by this project and their support scope.
- **Getting Started with Builds**: Provides baseline information for users who want to build Wine and DXMT themselves or understand the development environment.
- **Reference**: Adds background knowledge about the roles of Wine, DXMT, and Rosetta 2.
