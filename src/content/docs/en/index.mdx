## Documentation Overview

Bob-Ddong-Iri-Hoyo is a launcher designed to make it easier to run Eternal Return and HoYoverse games on macOS by managing Wine, DXMT, and Rosetta 2 configurations.
This documentation explains not only how to use the launcher, but also the installation flow and the technical background behind how it works.
