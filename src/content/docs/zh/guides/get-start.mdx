## 什么是 Bob-Ddong-Iri-Hoyo？

Bob-Ddong-Iri-Hoyo 是一个 Wine 启动器，用于在 macOS 上运行 Eternal Return 和 HoYoverse 游戏。
它内部使用基于 Wine 的运行环境，在 Apple Silicon Mac 上也可能配合 Rosetta 2、DXMT 等兼容层一起使用。

这个启动器的目标，是让用户不必直接处理复杂的 Wine 命令和配置文件，也能选择并管理运行游戏所需的 Bottle、Wine 版本和图形转换层。
熟悉 Wine 的用户仍然可以直接使用 CLI 环境；而普通用户可以通过启动器更轻松地完成同样的操作。

## 特性

它用于解决在传统 CrossOver 或 Wine 中运行 Eternal Return 时常见的渲染问题。
