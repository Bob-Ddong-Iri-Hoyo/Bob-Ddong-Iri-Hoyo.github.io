## 文档概览

Bob-Ddong-Iri-Hoyo 是一个启动器，用于在 macOS 上更方便地运行 Eternal Return 和 HoYoverse 游戏，并管理 Wine、DXMT、Rosetta 2 等组件。
本站不仅整理了基础用法，也会说明安装流程以及启动器背后的技术背景。
