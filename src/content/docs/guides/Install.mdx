## 설치

런처 앱은 Apple 개발자 미인증 프로그램입니다.(돈 없음 이슈) 따라서 실행 시 macOS Gatekeeper에 의해 차단될 수 있습니다. 이를 우회하려면 Apple의 quarantine 태그를 직접 제거해야 합니다.

터미널을 열고 아래 명령어를 입력하여 제거해야 합니다.

```bash
xattr -d com.apple.quarantine path/to/App.dmg
```
