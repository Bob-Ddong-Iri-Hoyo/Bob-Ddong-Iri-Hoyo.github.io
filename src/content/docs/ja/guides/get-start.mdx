## Bob-Ddong-Iri-Hoyo とは？

Bob-Ddong-Iri-Hoyo は、macOS で Eternal Return と HoYoverse ゲームを実行するための Wine ランチャーです。
内部では Wine ベースの実行環境を使用し、Apple Silicon Mac では Rosetta 2 や DXMT のような互換レイヤーがあわせて使われることがあります。

ランチャーの目的は、複雑な Wine コマンドや設定ファイルを直接扱わなくても、ゲーム実行に必要な Bottle、Wine バージョン、グラフィック変換レイヤーを選択して管理できるようにすることです。
Wine に詳しいユーザーは CLI 環境を直接使うこともできますが、一般的なユーザーはランチャーを通じて同じ作業をより簡単に行えます。

## 特徴

従来の CrossOver や Wine で Eternal Return を実行したときに発生しやすいレンダリング問題を解決します。

## ドキュメント構成

このドキュメントは、次の流れで読むことを想定しています。

- **インストール**: ランチャーをダウンロードし、macOS で実行できるように準備する手順を説明します。
- **Wine について**: このプロジェクトが提供する Wine ビルドと対応範囲を説明します。
- **ビルドを始める**: Wine と DXMT を自分でビルドする、または開発環境を理解したいユーザー向けの基準情報を整理します。
- **リファレンス**: Wine、DXMT、Rosetta 2 がどのような役割を持つのか、背景知識を補足します。
