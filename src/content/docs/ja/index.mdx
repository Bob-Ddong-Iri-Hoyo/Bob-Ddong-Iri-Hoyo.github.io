## ドキュメント概要

Bob-Ddong-Iri-Hoyo は、macOS で Eternal Return や HoYoverse ゲームを実行するために、Wine、DXMT、Rosetta 2 の構成を扱いやすくするランチャーです。
このドキュメントサイトでは、基本的な使い方だけでなく、ランチャーがどのような技術の上で動作しているのかも、インストール手順や開発背景とあわせて説明します。
